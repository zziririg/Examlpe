전역속성(Global Attributes)

class
공백으로 구분된 요소의 별칭 지정(중복가능)
id
문서에서 고유한 식별자를 정의(html당 1개만 부여,중복X)
style
요소에 적용할 CSS스타일 선언
title
요소의 정보(설명)을 지정, 요소에 마우스 오버하면 설명이 나옴

제목

Google
DIV
lang
요소의 언어(iso 639-1)를 지정, 주로 html태그에 사용
data-*
사용자 정의 데이터 속성을 지정
HTML에 JS에서 이용할 수 있는 데이터를 저장하는 용도로 사용

My data
위 'My data'에 내 '이름'과 '나이' 데이터를 저장해놓았음(콘솔확인)
draggable
요소의 drag and drop API를 사용 가능한지 여부 결정, drag&drop되게 할거냐 말거냐
hidden
요소를 숨김, form 태그에 자주 쓰임
tabindex
Tab 키를 이용해 요소를 순차적으로 포커스 탐색할 순서를 지정
대화형콘텐츠는 기본적으로 코드순서대로 탭 순서 있음
비대화형 콘텐츠에 tabindex="0" 지정하면 대화형콘텐츠처럼 탭순서 적용
